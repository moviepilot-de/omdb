mailinglisten @ omdb

Das omdb Projekt bietet eine Reihe von öffentlichen Diskussionslisten an. Um an den Diskussionen teilnehmen zu können, ist eine vorherige Anmeldung notwendig. Lediglich die Liste info-de erlaubt Nachrichten von Nicht-Mitgliedern.

info-de

Generelle Mailingliste für Diskussionen und Anmerkungen rund um das omdb Projekt.

content-de

Mailingliste zum Austausch von Informationen regelmäßiger Autoren von omdb.

tech-de

Für technische Diskussionen zur Software und zukünftigen Erweiterungen von omdb.

new-movies

Informationen über neu angelegte Filme auf omdb (in englisch - nur lesender Zugriff).
